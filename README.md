# sprd_bsp_sign

SPRD image signing script/program collection.
Please note that these programs from SPRD is buggy and vulnerable.

## `jingpad`
Taken from [`jingpad-bsp`](https://github.com/jingpad-bsp), for "modern" sprd devices using AVB for boot/recovery signing

## `kaios`
Taken from [`yonglongliu/vendor`](https://github.com/yonglongliu/vendor), for "legacy" sprd devices with boot/recovery signed using bsp sign (`DHTB`), likely android K-M
